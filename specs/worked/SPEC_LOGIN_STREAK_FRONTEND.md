# Спека (фронтенд): модальное окно серии входов → фри-спин

**Папка:** `specs/`
**Статус:** Draft
**Зависит от:** `SPEC_LOGIN_STREAK_FREE_SPIN_BACKEND.md` (реализован)

---

## Суть задачи

При каждом авторизованном входе пользователю показывается модальное окно «Ежедневная серия».
Оно демонстрирует прогресс трёхдневного цикла и мотивирует вернуться на следующий день.
На третий день подряд пользователь визуально «получает» фри-спин прямо в этом модальном окне.

---

## Когда показывать модальное окно

Модальное окно показывается **один раз за UTC-день** сразу после успешной авторизации.
Флаг «показали сегодня» хранится в `localStorage`:

```
KEY: streak_modal_shown_day
VALUE: "YYYY-MM-DD" (UTC)
```

Логика показа (выполняется при монтировании корневого layout после авторизации):

```typescript
const todayUtc = new Date().toISOString().slice(0, 10);
const shownDay = localStorage.getItem('streak_modal_shown_day');
if (shownDay !== todayUtc) {
  // показать модалку, затем:
  localStorage.setItem('streak_modal_shown_day', todayUtc);
}
```

## API

**Эндпоинт:** `GET /events/lucky-wheel/streak`

**Ответ:**

```typescript
interface LuckyWheelStreakRes {
  currentStreak: number;       // 1, 2, 3, 4, 5, ...
  daysUntilFreeSpin: number;   // 1 | 2 | 3  (сколько осталось до следующего фри-спина)
  freeSpinBalance: number;     // накопленные фри-спины
}
```

Запрос делается при монтировании layout (вместе с проверкой флага в localStorage).
Если эндпоинт вернул ошибку — модальное окно не показывается (silent fail).

## Структура модального окна

### Заголовок

```
🔥 День {currentStreak}
```

Если `currentStreak > 3`, то показываем общее количество:
```
🔥 День {currentStreak} — серия продолжается!
```

### Прогресс-трек (три дня)

Визуальный ряд из трёх ячеек:

```
[ День 1 ✓ ]  [ День 2 ✓ ]  [ День 3 🎁 ]
```

| Состояние ячейки | Условие | Внешний вид |
|---|---|---|
| Пройден | позиция ≤ `currentStreak % 3` (или `=== 0` для третьего) | фон акцентного цвета, галочка ✓ |
| Текущий | позиция === `currentStreak % 3` + 1 (если не кратный) | пульсирующий бордер |
| Предстоящий | позиция > текущего | серый, иконка замка |
| Наградной (день 3) | третья ячейка | иконка подарка 🎁, подпись «Фри-спин» |

Пример для `currentStreak = 2`:
```
[ ✓ День 1 ]  [ ✦ День 2 ]  [ 🔒 День 3 🎁 ]
                ^ активный
```

Пример для `currentStreak = 3` (кратный — получаем фри-спин):
```
[ ✓ День 1 ]  [ ✓ День 2 ]  [ 🎉 День 3 🎁 ]
```

Счётчик под треком: «Серия: {currentStreak} дн.»
Если `freeSpinBalance > 0`: дополнительная подпись «🎰 Фри-спинов на балансе: {freeSpinBalance}»

## Призыв к действию — тексты по дням

Призыв к действию зависит от позиции в текущем цикле (`currentStreak % 3`).

| `currentStreak % 3` | День цикла | Заголовок | Подзаголовок |
|---|---|---|---|
| `1` | День 1 | «Отличное начало!» | «Возвращайся завтра — ты уже на пути к бесплатному спину 🎰» |
| `2` | День 2 | «Ты в шаге от награды!» | «Ещё один день — и фри-спин у тебя в кармане. Так держать 💪» |
| `0` | День 3 🎁 | «Ты получил фри-спин! 🎉» | «3 дня подряд — это сила. Используй его на Lucky Wheel прямо сейчас!» |

Для длинных серий (`currentStreak > 3`) добавить к заголовку суффикс:

| `currentStreak` | Суффикс заголовка |
|---|---|
| 6, 9, 12 ... | «+ещё один фри-спин в копилку!» |
| все остальные > 3 | заголовок без изменений |

## Анимация на день 3 (получение фри-спина)

Условие: `currentStreak % 3 === 0`

Последовательность:
1. Модальное окно открывается — все три ячейки ещё серые.
2. Задержка 300 мс — первая ячейка «загорается» (анимация scale + fill).
3. Задержка 300 мс — вторая ячейка.
4. Задержка 300 мс — третья ячейка + конфетти/частицы по модальному окну (лёгкое, не навязчивое).
5. Подзаголовок меняется на «Ты получил фри-спин! 🎉».
6. Появляется кнопка «Крутить колесо».

Если `currentStreak % 3 !== 0` — ячейки отображаются сразу без анимации поочерёдного «загорания».

## Кнопки

### День 1 и День 2

```
[ Закрыть ]
```

Одна нейтральная кнопка. Никаких переходов — просто закрыть.

### День 3 (фри-спин получен)

```
[ Закрыть ]    [ 🎰 Крутить колесо ]
```

«Крутить колесо» — переход на страницу Lucky Wheel.

### Если `freeSpinBalance > 1` (накоплено несколько)

Добавить подсказку под кнопкой:
```
У тебя {freeSpinBalance} фри-спина(ов) — используй их в любой день
```

## Граничные случаи

| Ситуация | Поведение |
|---|---|
| Первый вход в приложение | `currentStreak = 1` → показываем «Отличное начало!» |
| Пользователь пропустил день | Серия сбросилась до `1` — показываем день 1 без упоминания сброса |
| `currentStreak = 6` | `6 % 3 === 0` → анимация + «+ещё один фри-спин в копилку!» |
| API недоступен при открытии | Модальное окно не показывается (не ломать UX) |
| Модальное окно открыто → пользователь перезагрузил страницу в тот же день | `localStorage` содержит today → не показывать повторно |

## RTK Query

```typescript
// shared/api/lucky-wheel.api.ts
export const luckyWheelApi = createApi({
  reducerPath: 'luckyWheelApi',
  baseQuery: fetchBaseQuery({ ... }),
  endpoints: (builder) => ({
    getStreak: builder.query<LuckyWheelStreakRes, void>({
      query: () => '/events/lucky-wheel/streak',
    }),
  }),
});

export const { useGetStreakQuery } = luckyWheelApi;
```

В `useLoginStreakModal`:

```typescript
const todayUtc = new Date().toISOString().slice(0, 10);
const alreadyShown = localStorage.getItem('streak_modal_shown_day') === todayUtc;

const { data, isSuccess } = useGetStreakQuery(undefined, {
  skip: alreadyShown,  // не делать запрос если уже показали
});

useEffect(() => {
  if (isSuccess && data && !alreadyShown) {
    setOpen(true);
    localStorage.setItem('streak_modal_shown_day', todayUtc);
  }
}, [isSuccess, data]);
```
